# MarketSentinel

Event-Driven Stock Movement Prediction & Trading Signal Platform

## Description
MarketSentinel analyzes financial news sentiment and historical stock data
to predict short-term stock movement and generate BUY/SELL/HOLD signals.

## Tech Stack
- FastAPI
- XGBoost, LSTM, Prophet
- FinBERT (Sentiment Analysis)
- Docker, GitHub Actions
- Prometheus & Grafana
